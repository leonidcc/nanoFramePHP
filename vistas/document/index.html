Instalación

Abrir el phpMyAdmin y crear una nueva base de datos importando el archivo Template.sql o crear la base de datos Template, modificar el archivo config.php

Ir a la carpeta del proyecto y abrir una terminal de linux o window

User@pc: ~/nanoFrameworkPhp$

Para isntalalr los valores por defecto de la base de datos ejecutar

User@pc: ~/nanoFrameworkPhp$ php gen/installdb.php

Ejecutar para iniciar un servidor con el proyecto

User@pc: ~/nanoFrameworkPhp$ php -S localhost:3000

Ir al navegador a la url http://localhost:3000/

Estructura del proyecto MVC

@recursos

Imagenes, iconos generales para el entorno

Nucleo del nanoframework, no se suele modificar

Librerias por defecto del nanoframework

gen

Script que te permite generar código Crud rapidamente, intententar

User@pc: ~/nanoFrameworkPhp$ php core/crud.php tablatest name:string edad:int

Levantar el servidor e ir a tablatest para ver el resultado
